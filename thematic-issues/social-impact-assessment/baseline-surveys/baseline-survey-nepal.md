# baseline surveys - Nepal


I've been working with [district-level socio-economic profiles](https://cbs.gov.np/district-profile/) in Nepal. Nepal's [Central Bureau of Statistics](https://cbs.gov.np) has a lot of great materials on its website, but its interface makes it difficult to quickly find and access data for a particular district. I built this index to help expedite my searches and am sharing it in case anyone else is looking.
